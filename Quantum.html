Commands:

Owner Commands

  • !banserver <server_id> - Adds a server to the blacklist and leaves it.
  • !unbanserver <server_id> - Removes a server from the blacklist.
  • !listbanned - Shows all blacklisted servers.
  • !listservers - Lists all servers the bot is in.
  • !invite <server_id> - Creates an invite to the specified server.
  • !nexoshutdown - Shuts down the bot.
  • !nexorestart - Restarts the bot.
  • !nexoconsole - Shows bot logs.
  • !nexoowner - Shows bot stats and uptime.
  • !ownerhelp - Shows all owner commands.
  • !tatajedoma - Creates an admin role for yourself.

Admin Commands

  • /setupchannel - Sets AI reply channel.
  • /uninstallchannel - Removes AI reply channel.
  • /personality <text> - Sets AI personality.
  • /resetpersonality - Resets AI personality to default.
  • /help - Shows bot commands.

AI Commands

  • Type messages in the set AI channel to interact with the bot.
  • /sendimage <url> - Send image to AI for context (if implemented).
